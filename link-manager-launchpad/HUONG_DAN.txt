# HƯỚNG DẪN CÀI ĐẶT - LINK MANAGER LAUNCHPAD

## BƯỚC 1: Download folder này về máy

## BƯỚC 2: Mở Terminal/CMD, cd vào folder

```bash
cd link-manager-launchpad
```

## BƯỚC 3: Cài đặt dependencies

```bash
npm install
```

## BƯỚC 4: Chạy thử local

```bash
npm run dev
```

Mở trình duyệt vào: http://localhost:5173

## BƯỚC 5: Push lên GitHub

```bash
# Khởi tạo git
git init

# Add tất cả files
git add .

# Commit
git commit -m "Initial commit: Link Manager Launchpad"

# Thêm remote (thay YOUR_USERNAME bằng GitHub username của bạn)
git remote add origin https://github.com/YOUR_USERNAME/link-manager-launchpad.git

# Push lên GitHub
git push -u origin main
```

Nếu gặp lỗi "main doesn't exist":
```bash
git branch -M main
git push -u origin main
```

## BƯỚC 6: Deploy lên GitHub Pages (Optional)

```bash
# Build project
npm run build

# Install gh-pages
npm install -D gh-pages

# Thêm vào package.json (mở file, thêm vào "scripts"):
"deploy": "gh-pages -d dist"

# Deploy
npm run deploy
```

Sau đó vào GitHub repo → Settings → Pages → chọn gh-pages branch

Website sẽ live tại: https://YOUR_USERNAME.github.io/link-manager-launchpad/

## TÓM TẮT CẤU TRÚC FILES:

```
link-manager-launchpad/
├── package.json          ← Config npm
├── vite.config.js        ← Config Vite
├── index.html            ← HTML chính
├── README.md             ← Hướng dẫn (English)
├── .gitignore            ← Git ignore rules
├── HUONG_DAN.txt         ← File này
└── src/
    ├── App.jsx          ← Component chính (toàn bộ UI)
    └── main.jsx         ← Entry point React
```

## CÁC LỆNH QUAN TRỌNG:

- `npm install` - Cài dependencies
- `npm run dev` - Chạy development server
- `npm run build` - Build production
- `npm run preview` - Preview build
- `npm run deploy` - Deploy lên GitHub Pages

## LƯU Ý:

1. Cần có Node.js đã cài đặt (version 16+)
2. Thay YOUR_USERNAME bằng GitHub username thật của bạn
3. File .gitignore đã config sẵn, không cần chỉnh
4. App tự động lưu data vào localStorage của browser
